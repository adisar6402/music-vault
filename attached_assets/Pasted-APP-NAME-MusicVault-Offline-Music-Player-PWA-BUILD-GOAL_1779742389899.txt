APP NAME:

MusicVault — Offline Music Player PWA

🧠 BUILD GOAL:

Build a modern, premium-looking Offline Music Vault PWA that lets users:

upload music files
play music offline
create playlists
enjoy smooth Spotify-like UI/UX
install as a PWA on mobile/desktop

The app must feel like a real production mobile app, not a basic website.

No backend required. Use browser storage only.

⚙️ TECH STACK
React (or Next.js if needed)
TailwindCSS for styling
IndexedDB or localStorage for saving songs/playlists
PWA support (service worker + manifest)
No backend
📱 CORE FEATURES
1. 🎵 Music Upload System
Users can upload MP3 files
Store files locally in browser (IndexedDB preferred)
Show uploaded songs in library instantly
2. 🎧 Music Player (Premium UX)

Build a full-screen now playing screen:

Must include:
Large album art placeholder (auto-generated gradient if none)
Song title + artist name
Waveform-style progress bar (animated)
Play / Pause button (center big button)
Next / Previous buttons
Shuffle + Repeat toggle
Volume slider
Gestures (VERY IMPORTANT):
Swipe left → next song
Swipe right → previous song
Tap → play/pause
3. 📁 Library Screen
Grid + list view toggle
Recently added section
Search bar (instant filter)
Favorite songs ❤️ system
4. 🎼 Playlist System
Create playlist
Add/remove songs
Rename playlist
Delete playlist
Playlist detail screen
5. 💜 Mood Mode (IMPORTANT UX FEATURE)

Add a “Mood Mode” selector:

😌 Chill
🧠 Focus
🔥 Workout
🌙 Sleep

This filters playlists or songs visually (no AI needed).

6. 🔥 Viral Feature — Share Playlist Link

Generate shareable playlist links like:

/playlist/relax123

When opened:

shows playlist preview
allows user to import songs manually

This is for virality via WhatsApp sharing.

📲 PWA REQUIREMENTS (VERY IMPORTANT)

Make it fully installable:

Must include:
manifest.json
service worker
offline caching support
“Add to Home Screen” prompt

App must work fully offline after first load.

🎨 UI / UX DESIGN (TOP QUALITY — SPOTIFY LEVEL)
Visual Style:
Dark mode default (pure black background)
Accent color: neon purple or neon green
Glassmorphism cards (blur + transparency)
Smooth animations (fade, slide, scale)
Layout style:
Home screen:
Top greeting: “Welcome back 🎧”
Recently played carousel
Big “Add Music” button
Playlists preview cards
Player screen:
Full screen focus UI
blurred background from album art
centered play button
waveform progress bar
minimal distractions
Animations:
smooth page transitions
micro-interactions on buttons
floating mini-player when navigating
⚡ PERFORMANCE RULES
app must be fast (<1s interactions)
lazy load audio files
avoid heavy libraries
smooth scrolling always
🧠 DATA STRUCTURE (LOCAL ONLY)

Songs:
{
id,
title,
artist,
file,
duration,
favorite
}

Playlist:
{
id,
name,
songs[],
createdAt
}

🚫 IMPORTANT CONSTRAINTS
No backend
No login system
No external APIs required
Must work fully offline after install
Must be simple but premium feeling
🧪 FINAL OUTPUT EXPECTATION

The final app should feel like:

“A lightweight Spotify-style offline music app you install on your phone and actually use daily.”
